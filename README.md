# مهندسی نرم افزار

### فصل یازدهم - طراحی واسط کاربر

در این پروژه بخش طراحی واسط برنامه های تحت وب مورد بحث قرار میگیرد.

### نصب
برای مشاهده و اجرا، پروژه مورد نظر را از حالت فشرده خارج کرده و فایل با نام index را در مرورگر خود باز کنید.

### منابع

تصاویر استفاده شده در این پروژه از [گوگل](https://google.com) کپی شده اند.
 
 طراحی و پیاده سازی سبک اسلایدوار از کتابخانه ی [impress.js](https://github.com/impress/impress.js) برگرفته شده است ولی طبق نیاز بنده شخصی سازی شده است.
